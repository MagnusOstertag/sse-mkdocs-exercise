Documentation Example

This repository contains some example C++ code from the Simulation Software Engineering lecture. It is used in the exercise to generate a documentation website. For more information please check out the exercise sheet.

The code is a slightly modified version of the code used in the CMake exercise and thus also similar to the code from the Spack exercise.
